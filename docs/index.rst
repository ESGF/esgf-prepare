``esgprep`` toolbox
===================

The ESGF ``esgprep`` toolbox is a piece of software that enables data preparation according to ESGF best practices.
It can be used to fetch required confioguration files, apply the Data Reference Syntax on local filesystems or generate
mapfiles for ESGF publication.
